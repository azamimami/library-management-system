# 📖 Hakkında

Bu proje, kütüphanelerin günlük işlemlerini dijital ortama taşıyan kapsamlı bir masaüstü uygulamasıdır. Kitap takibi, üye yönetimi, ödünç işlemleri ve detaylı raporlama özellikleri ile kütüphane yönetimini kolaylaştırır.

# ✨ Özellikler

# 📚 Kitap Yönetimi
- Kitap ekleme, düzenleme, silme
- Kategori bazlı filtreleme
- Gelişmiş arama (ad, yazar, kategori)
- Kitap detayları (yıl, sayfa, dil)

### 👥 Üye Yönetimi
- Üye ekleme, düzenleme, silme
- Otomatik üye numarası (LIB000001)
- Üye arama

# 📖 Ödünç İşlemleri
- Kitap ödünç verme
- Kitap iade alma
- Gecikme ücreti (günde 5 TL)
- Gecikmiş kitap uyarısı

# 📊 İstatistikler
- Grafiksel gösterim
- En çok okunan kitaplar
- En aktif üyeler
- Kapsamlı rapor

# 👤 Kullanıcı Yönetimi
- Admin ve personel rolleri
- Güvenli giriş sistemi
